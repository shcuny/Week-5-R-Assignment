Both, appending and preallocating are ways to add elements to the vectors, dataframes and lists. You can build a data frame by appending new rows one by one which is okay for small data frame but this method is memory intensive. For bigger data frame, you should preallocate space for the vectors, dataframes, and list for the type data and insert values into the vectors of the type you want for each column and lastly create your dataframe.
This method is more efficient but the downside is that you need to know the number of rows.

I got the example from the following link: http://stackoverflow.com/questions/23985896/append-to-a-vector-quick-method

using preallocation

append3 <- function(n) { v <- numeric(n) for (i in 1:n) { v[i] <- i } v }

using append

append4 <- function(n) { v <- numeric() for (i in 1:n) { v <- append(v, i) } v }
